## What’s new in MoBud v0.000.026
This update brings a range of improvements to make everyday use smoother, clearer and more reliable.

## Faster and safer trip registration
Both dashboard quick-log and trip registration have matured:

	# Dashboard:
	- selecting a vehicle no longer registers a trip immediately;
	- a trip is only saved after pressing Register;
	- the selected vehicle is now highlighted correctly;

	# Better routes with multiple stops
	- address suggestions also work for intermediate stops;
	- selected addresses are saved correctly for route calculation;
	- the latest search result is always used while typing.


## Clearer backup settings
The storage section has been reorganised into two clear parts:

	# Local data
	- export a local backup;
	- import a local backup;
	- keep a JSON backup file on your device or another safe location.

	# Google Drive
	- export your data to Drive;
	- import a backup from Drive;
	- merge data to reduce the risk of missing trips, receipts or other records.
	- Vehicle filter in Calendar

	# Limit token availability:
	Google Drive now works as an intentional backup and transfer option only instead of continuous background sync.
	- Export to Google Drive
	- Import from Google Drive
	- Combine local data with an existing backup;
	- Move your MoBud data more safely between devices.

Google Drive access is only requested when you choose to import or export a backup with a temporary token.


## Calendar improvements.
	You can filter your view:
	- all vehicles;
	- one selected vehicle;
	- trip totals and day details for that vehicle only.

	Calendar dates now have a more consistent visual state:
	- selected dates use a cleaner, more subtle outline;
	- dates with activity remain easy to identify.

# Improved Reports layout
	Reports have received several visual improvements:
	- Total trips, Total costs and Total receipts now use the full desktop width;
	- long vehicle names stay on one line and are shortened with ...;
	- the previous eye emoji has been replaced with a clean custom eye icon.

	Improved print-layout;

# Improved FAQ and translations
The in-app and public FAQ have been updated:
- Google Drive information now reflects the new backup flow;
- links display correctly instead of showing raw HTML;
- newly added interface text has been reviewed across the available languages;
- the tagline “Track every ride. Own every kilometer.” remains in English.
- Updated homepage and production app
